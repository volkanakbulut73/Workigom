1. **Go to Render Dashboard**
   - Visit: https://dashboard.render.com/
   - Sign in or create an account

2. **Deploy with Blueprint**
   ```
   1. Click "New +" ‚Üí "Blueprint"
   2. Select: volkanakbulut73/Workigom
   3. Render will detect render.yaml
   4. Click "Apply"
   5. Wait 5-10 minutes for deployment
   ```

3. **Run Database Migrations**
   ```bash
   # After backend is deployed, open backend shell and run:
   npx prisma migrate deploy
   ```

4. **Update Environment Variables**
   - Backend: Update `CORS_ORIGIN` with your frontend URL
   - Frontend: Update `VITE_BACKEND_URL` with your backend URL

5. **Redeploy Services**
   - Services will auto-redeploy after env var changes
